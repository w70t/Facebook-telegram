# Facebook ⇄ Telegram — بوت نسخ ونشر (يُدار بالكامل من تلغرام)

ينسخ منشورات **قناة تلغرام** (حتى لو لست أدمن فيها) عبر **حسابك الشخصي**،
يرسلها إلى **بوت مراجعة** فيه أزرار، وبعد موافقتك يُنشر على **صفحة فيسبوك**.
كل الإعداد يتم **من داخل تلغرام** — بدون تعديل أي ملفات. مصمّم للعمل على **Raspberry Pi 5**.

```
قناة تلغرام ─▶ حسابك الشخصي ─▶ بوت المراجعة (أزرار) ─▶ موافقتك ─▶ صفحة فيسبوك
```

> 📘 دليل الإعداد خطوة بخطوة: **[SETUP_AR.md](SETUP_AR.md)**

## المميزات
- كل شيء من تلغرام: **تسجيل الدخول بالرقم، فيسبوك، القنوات، حسابات X، الأدمنون** — عبر أزرار.
- مصدران: **قنوات تلغرام** + **حسابات X (تويتر)** — كلاهما يدخل نفس خط المراجعة.
- تسجيل دخول الحساب برقم الهاتف **مع رمز الدولة أو بدونه** (رمز افتراضي قابل للضبط).
- إدارة **عدة قنوات وعدة حسابات X** من داخل البوت.
- **فلترة كلمات**: أي منشور يحتوي كلمة ممنوعة يُتجاهل تلقائياً (تلغرام و X).
- **X: تغريدات فقط** — يتجاهل الردود والريتويت (قابل للتبديل).
- أزرار مراجعة لكل منشور: **✅ نشر** / **📄 نشر النص فقط** / **✏️ تعديل** / **❌ تجاهل**.
- زر **🔄 تحديث من GitHub** يحدّث الكود ويعيد التشغيل على الـ Pi.
- كل الإعدادات تُحفظ في `settings.json` وتبقى بعد إعادة التشغيل.

## البدء السريع
```bash
git clone https://github.com/w70t/Facebook-telegram.git
cd Facebook-telegram
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

python setup.py     # مرة واحدة: api_id + api_hash + توكن البوت
python main.py      # شغّل البوت
```
ثم افتح البوت في تلغرام:
1. أرسل `/claim الرمز` (الرمز يظهر في سجل التشغيل) → تصبح المالك.
2. أرسل `/panel` → تظهر لوحة التحكم.
3. اضبط بالترتيب: 🔐 تسجيل الدخول ← 📘 فيسبوك ← 📍 قروب المراجعة ← 📡 القنوات.

## لوحة التحكم `/panel`
| الزر | الوظيفة |
|------|---------|
| 🔐 تسجيل دخول الحساب | إدخال رقم الهاتف والرمز داخل تلغرام |
| 📘 إعداد فيسبوك | معرّف الصفحة + توكن النشر |
| 📍 تعيين قروب المراجعة | يجعل المحادثة الحالية مكان المراجعة |
| 📡 قنوات تلغرام المصدر | إضافة/حذف/عرض القنوات |
| 🐦 حسابات دخول X | مجموعة حسابات دخول: يبدّل تلقائياً لو انحظر أحدها |
| 🐦 حسابات X المتابَعة | إضافة/حذف حسابات تويتر + تبديل تجاهل الردود |
| 🚫 فلترة الكلمات | كلمات ممنوعة تُتجاهل منشوراتها (تلغرام و X) |
| 👤 الأدمنون | إضافة/حذف أدمنين (للمالك) |
| 🌍 رمز الدولة الافتراضي | يُستخدم عند إدخال رقم بدون رمز دولة |
| 🔄 تحديث من GitHub | `git pull` + إعادة تشغيل على الـ Pi |
| ℹ️ الحالة | ملخّص ما تم إعداده |

## التشغيل الدائم على Raspberry Pi
```bash
sudo cp deploy/telegram-fb-bot.service /etc/systemd/system/
sudo nano /etc/systemd/system/telegram-fb-bot.service   # عدّل User والمسارات
sudo systemctl daemon-reload
sudo systemctl enable --now telegram-fb-bot
journalctl -u telegram-fb-bot -f
```

## تنبيهات
- ⚠️ استخدم **رقماً/حساباً ثانوياً** للحساب الشخصي (أتمتة الحساب مخالفة لشروط تلغرام).
- ⚠️ قراءة X تتم بطريقة **غير رسمية** (twikit) — مخالفة لشروط X وفيها خطر حظر الحساب. استخدم **حساب X ثانوياً**.
- ⚠️ احترم حقوق المحتوى وأضف مصدراً عند اللزوم.
- ملف `settings.json` يحتوي مفاتيح حسّاسة — **لا يُرفع إلى GitHub** (مستثنى في `.gitignore`).
